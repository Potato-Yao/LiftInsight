## recovery time

hard set: $RPE \geq 6$, or $RIR \leq 4$

| RPE | Weight |
| --- | -----: |
| 6   |      1 |
| 7   |      2 |
| 8   |      3 |
| 9   |      5 |
| 10  |      8 |

| Type                                     | Factor |
| ---------------------------------------- | -----: |
| Isolation                                |      1 |
| Machine compound                         |    1.5 |
| Free-weight compound                     |      2 |
| Axial fatigue monsters (deadlift, squat) |      3 |

$$
\text{fatigue} = \sum (\text{hard sets} \times \text{RPE weight} \times \text{factor})
$$

| Fatigue Score | Recovery Need         |
| ------------- | --------------------- |
| <20           | Easy                  |
| 20-40         | 1 day                 |
| 40-70         | 2 days                |
| 70-100        | 3 days                |
| 100+          | You fought a forklift |

## total intensity

$$
\text{work load} = \text{session RPE} \times \text{Duration in min}
$$
